SOUTHCARAVAN HOME PAGE OUTLINE (/)
===================================

The home page is a marketplace-first landing experience at route `/`. It is wrapped by the public app shell (Header, Marketplace Action Strip, Footer) and renders product discovery content in a single scrollable column (max width ~1500px).

---

1. GLOBAL SHELL (AppShell — wraps all public pages including home)

   A. Header (sticky on desktop)
      - Left: workspace menu (dashboard consoles only), categories sidebar toggle (hamburger), SouthCaravan logo + wordmark
      - Center/right: search bar, “Post RFQ” button, messages link, currency selector, cart button, profile menu (sign-in/join or signed-in avatar)
      - Mobile: compact top bar with logo, search, cart, profile; secondary row for key actions
      - Categories sidebar (AddItemsSidebar): opens on menu hover/click for browsing taxonomy

   B. Marketplace Action Strip (home page only, sticky below header on md+)
      - Quick links: Categories, Featured, Services, Post RFQ
      - Horizontally scrollable category pills (up to 14 visible) linking to `/categories?category=…`
      - “+N” overflow and “All” search open a category command dialog

   C. Footer
      - Brand block: logo, site description, social links (LinkedIn, Twitter, Facebook)
      - Link columns: Explore (Marketplace, FAQ), Company (About, Blog, Careers, Contact), Legal (Privacy, Terms, Cookies, Compliance)
      - Contact row: email, phone, office address (Kampala, Uganda)
      - Copyright bar with Privacy, Terms, Contact links

---

2. MAIN CONTENT (app/page.tsx)

   A. Featured Picks Card (hero product block)
      - Card header: “Featured picks” with link to `/featured`
      - Layout: 2-column grid on md+
        • Left (spotlight): large image, “Spotlight” badge, product name, category badge, price — links to product detail
        • Right: 2×2 grid of up to 4 additional featured products (image, name, category, price)
      - Data: first 5 featured products (or fallback landing products if none featured)

   B. Sponsored / Product Ad Banner Section
      - Rotating promo banner carousel (up to 5 sponsored items, auto-rotates every ~7.5s)
      - Each slide: “Featured ad” + “Verified suppliers” badges, product name, headline, price, category pill, CTA buttons (“View product”, “See details”), large banner image
      - Dot indicators and “Banner X of Y” when multiple items
      - Empty state: “Sponsored placements — No sponsored placements are available right now.”

   C. More Featured (conditional)
      - Shown only when more than 5 featured products exist
      - Section title “More featured” with “View all” link to `/featured`
      - Responsive product grid (up to 12 cards) using CategoryProductCard

   D. Category Infinite Feed (infinite scroll)
      - Initial server load: 3 category sections, 5 products per section
      - Each section: bordered card with category title (links to filtered categories page), “All” link, horizontal product row
      - Products rendered as CategoryProductCard (image, featured badge, name, price, supplier label; supports products and services)
      - Loads more via IntersectionObserver → `/api/landing/products` (3 categories per page)
      - States: loading spinner, error card, empty message, “End of list” when exhausted

---

3. PRODUCT CARD PATTERN (used throughout home)

   CategoryProductCard:
   - Square image (or placeholder icon)
   - Optional “Featured” or “Service” badge
   - Product name (2-line clamp), price with currency conversion, supplier/subcategory line
   - Links to `/product/{id}` or service href

---

4. DATA & SEO

   - Server-fetched: featured products (up to 60), sponsored items, initial category feed sections
   - Fallback: if no featured products, uses general landing products for hero
   - Metadata: site home title, default description, brand/Uganda/B2B keywords
   - HomePageJsonLd structured data injected for search engines

---

5. VISUAL / LAYOUT NOTES

   - Full-page min-height background
   - Top section uses productPageInsetClassName padding; content centered in max-w-[1500px]
   - Marketplace-focused (not a traditional marketing hero with stats/testimonials)
   - Primary user flows: browse featured → view product, browse by category, post RFQ, search, cart/checkout
